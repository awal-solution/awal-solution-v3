## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/awal-solution/awal-solution-v3.git
cd awal-solution-v3
yarn
```

## Start development

```sh
yarb dev
```

## Build for production

```sh
yarb build
```

## Lint with

```sh
yarb lint
```
